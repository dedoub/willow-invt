**인원**: 4명 (어머니, 본인, 딸 이이, 조카)
**항공**: 대한항공 (인천 ↔ 후쿠오카, 약 1시간 30분)

## 추천 일정

### Day 1 — 후쿠오카 시내 (도착일, 여유롭게)

| 시간 | 일정 | 메모 |
|------|------|------|
| 오전 | 인천 → 후쿠오카 도착 | 비행 1시간 30분 |
| 점심 | 하카타역 식당가 | 하카타 라멘 (이치란, 잇푸도 등) |
| 오후 | 캐널시티 하카타 | 쇼핑 + 분수쇼 (아이들이 좋아함) |
| 저녁 | 나카스 포장마차 거리 | 야타이에서 꼬치, 라멘 |
| 밤 | 호텔 체크인 & 휴식 | |

### Day 2 — 다자이후 + 온천 (하이라이트)

| 시간 | 일정 | 메모 |
|------|------|------|
| 오전 | 다자이후 텐만구 | 학문의 신사, 평지 위주로 어머니도 편안 |
| - | 참배길 산책 | 우메가에모치(매화떡) 필수 |
| 점심 | 다자이후 참배길 맛집 | |
| 오후 | 온천 (나카가와 세이류 등) | 노천온천 추천, 셔틀버스 운행 |
| 저녁 | 하카타역 복귀 → 모츠나베 | 곱창전골 (후쿠오카 3대 음식) |

> 유후인은 버스 편도 2시간이라 어머니/아이 체력 고려 시 다자이후 + 온천 조합이 더 적합

### Day 3 — 쇼핑 & 출발

| 시간 | 일정 | 메모 |
|------|------|------|
| 오전 | 오호리 공원 산책 | 호수 둘레길, 유모차/어르신 모두 OK |
| 오전~점심 | 텐진 지하상가 쇼핑 | 일본 최대 규모 지하상가 |
| 점심 | 텐진 맛집에서 마지막 식사 | |
| 오후 | 후쿠오카 → 인천 출발 | |

## 가족 구성 맞춤 팁

- **어머니**: 다자이후, 오호리공원은 평지 위주로 부담 없음. 온천 꼭 포함 추천
- **딸 이이**: 캐널시티 분수쇼, 다자이후 참배길 간식 코스
- **조카**: 캐널시티 내 가챠, 캐릭터샵 다수
- **숙소**: 하카타역 도보 5분 이내 추천 (이동 동선 최소화)

---

## 숙소 추천 지역

하카타역 주변이 교통/식사/쇼핑 모두 편리. 추천 호텔 타입:
- **비즈니스 호텔 2실**: 도미인 하카타 (온천 포함), 호텔 포르자 하카타
- **패밀리룸**: JR큐슈호텔 블라섬 하카타 (넓은 객실)

## 후쿠오카 필수 먹거리

1. **하카타 라멘** — 이치란, 잇푸도, 신신라멘
2. **모츠나베** (곱창전골) — 이치류, 야마야
3. **야타이 포장마차** — 나카스 강변
4. **우메가에모치** — 다자이후 참배길 매화떡
5. **멘타이코** (명란) — 후쿠야, 야마야 (기념품으로도 추천)
