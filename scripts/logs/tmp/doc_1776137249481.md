# Monitoring Port Operations Guide

모니터링 포트 운영 가이드 — 50만원으로 텐배거 후보를 걸러내는 방법

---

## 개요

모니터링 포트는 메인 피라미딩에 진입하기 전, 소액(50만원)으로 종목의 추세를 검증하는 사전 단계다. 메인 포트와 동일한 룰을 1/10 스케일로 적용한다.

**핵심 역할:** 50만원짜리 실험으로 500만원짜리 확신을 만든다.

| 항목 | 모니터링 포트 | 메인 포트 |
|------|-------------|----------|
| 트랜치 금액 | 50만 | 500만 |
| 최대 트랜치 | 10회 | 10회 |
| 풀 포지션 | 500만 | 5,000만 |
| 평균 수익률 기준 | 동일 | 동일 |
| BUY/FREEZE 룰 | 동일 | 동일 |
| SELL | PROMOTE 시 전량 매도 | 하우스 머니 / 논문 파기 |

---

## 매수 스케줄

평균 수익률이 아래 기준에 도달하면 50만원을 추가 매수한다.

| 트랜치 | 평균 수익률 | 스텝 | 누적 투입 |
|--------|-----------|------|----------|
| M1     | —         | 진입 | 50만     |
| M2     | +10%      | +10  | 100만    |
| M3     | +20%      | +10  | 150만    |
| M4     | +30%      | +10  | 200만    |
| M5     | +40%      | +10  | 250만    |
| M6     | +55%      | +15  | 300만    |
| M7     | +75%      | +20  | 350만    |
| M8     | +100%     | +25  | 400만    |
| M9     | +135%     | +35  | 450만    |
| M10    | +175%     | +40  | 500만    |

**스텝 패턴:** 10 → 10 → 10 → 10 → 15 → 20 → 25 → 35 → 40

---

## FREEZE

메인 포트와 동일. 직전 트랜치 기준 밑으로 내려가면 추가매수를 중단하고 방치한다.

- M3까지 넣었는데 +20% 밑으로 내려감 → 추가매수 중단. 방치.
- 다시 +20%를 회복하면 → 피라미딩 재개. 다음 목표 +30%.
- 영영 회복 못 하면 → 그냥 놔둠. 최대 150만 묶임.

모니터링 포트에서는 손절 매도를 하지 않는다.

---

## PROMOTE — 메인 포트 승격

### 전환 조건

M10까지 기다릴 필요 없다. 확신이 서면 승격한다.

| 모니터링 도달 | 확신 수준 | 승격 판단 |
|-------------|----------|----------|
| M3 (+20%)   | 초기 확인 | 추세가 안정적이면 전환 가능 |
| M5 (+40%)   | 중간 확인 | 강한 전환 시그널 |
| M7 (+75%)   | 높은 확인 | 거의 확정적 전환 |
| M10 (+175%) | 완전 확인 | 의심의 여지 없음 |

### 전환 절차

1. **메인 포트에서 T1 매수** — 500만원. 이 시점의 시장가로 매수.
2. **모니터링 포트 해당 종목 전량 매도** — 수익이든 손실이든 정산.
3. **메인 포트에서 T2~T10 피라미딩 진행** — T1 매수가 기준으로 새 사이클 시작.

### 주의사항

- 메인 T1의 매수가는 모니터링 진입가와 다르다. 메인 포트의 평균 수익률은 T1부터 새로 계산된다.
- 모니터링 포트 매도 대금은 메인 T1 매수 자금으로 사용해도 되고, 별도로 관리해도 된다.
- 어닝 발표 1주일 이내에는 PROMOTE도 보류한다. 발표 후 진행.

## 운영 규모와 자본 관리

### 추천 규모

- 동시 모니터링: 10~20종목
- M1 초기 자본: 500~1,000만원 (10~20종목 × 50만)
- 이 금액은 "종목 발굴 비용"으로 간주한다.

### 자본 흐름 시나리오 (15종목 시작 기준)

```
M1 진입: 15종목 × 50만 = 750만

3개월 후:
  8종목 FREEZE (M1~M2) → 400~800만 묶임
  4종목 M3~M4 진행 중 → 600~800만 투입됨
  2종목 PROMOTE → 메인 T1 진입 (각 500만)
  1종목 M5+ 진행 중 → PROMOTE 대기

6개월 후:
  메인 포트에서 1종목 T3~T5 진행 → 1,500~2,500만 투입
  나머지 1종목 FREEZE
  모니터링 포트에서 신규 종목 발굴 → M1 재투입
```

### 모니터링 포트의 총 비용

FREEZE된 종목에 묶이는 자본의 기대값:

- M1에서 FREEZE: 50만 × ~5종목 = 250만
- M2~M3에서 FREEZE: 100~150만 × ~3종목 = 300~450만
- 기대 총 묶임: **550~700만**

이 금액은 텐배거 1개(~2.9억)에 비하면 약 2~2.5% 수준이다.

---

## 모니터링 포트 DAILY ROUTINE

1. 증권앱에서 각 모니터링 종목의 **평균 수익률** 확인
2. 다음 M 트랜치 기준 도달? → **50만 추가 매수**
3. 승격 기준(M3~M5+) 도달 + 확신? → **PROMOTE 절차 실행**
4. 직전 트랜치 기준 밑? → **아무것도 안 함**
5. 해당 없으면? → **앱 닫기**

### 주간 리뷰 (매주 1회)

- 모니터링 포트 전체 현황 점검
- FREEZE 종목 중 회복 조짐 있는 종목 확인
- 신규 종목 발굴 → 빈 슬롯에 M1 진입
- 모니터링 + 메인 합산 자본 사용량 확인

## 종목 발굴 기준 (M1 진입 전)

모니터링 M1에 넣을 종목을 고르는 기준. 이것은 피라미딩 룰 밖의 정성적 판단이다.

### 필수 조건

- 미국 상장 기술주, 시가총액 $1B~$20B
- 최근 분기 매출 YoY 성장률 30% 이상
- 흑자 전환 완료 또는 임박 (영업이익률 개선 추세)
- 뚜렷한 경쟁 해자 (기술, 네트워크 효과, 전환 비용 중 1개 이상)

### 우대 조건

- 창업자가 CEO로 재직 중
- TAM이 아직 초기 침투 단계 (<10% 시장 점유)
- 최근 1년 내 주요 제품 출시 또는 시장 확장
- 기관 투자자 보유 비중 증가 추세

### 제외 조건

- 적자폭 확대 중
- 경영진 이슈 (SEC 조사, 내부자 대량 매도)
- 단일 고객 매출 의존도 50% 이상
- 기술적 해자 없이 가격 경쟁에 노출
